# Jenkins Job 생성 방법

#### 1. Jenkins 대시보드에서 **새로운 Item**을 클릭합니다.

![jenkinsJob-create-1](/uploads/760d30879db7eec2de95653dce81ad47/jenkinsJob-create-1.png)
