# Jenkins Plugin 설치 방법

1. Jenkins 대시보드에서 Jenkins 관리로 들어갑니다.

2. Plugin 탭을 클릭합니다.

3. Available(사용 가능한 플러그인) 탭에서 원하는 Plugin을 검색 후 설치를 진행합니다다.
