# Jenkins Job 생성 방법

1. Jenkins 대시보드에서 새로운 Item을 클릭한다.

2. Pipeline의 이름을 설정 후 'Freestyle project' 또는 'Pipelie'을 클릭 후 OK 버튼을 누르면 생성된다.
